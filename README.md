# RappiPlus-De-Datos-a-Decisiones-de-Negocio

## Resumen

Este proyecto analiza el desempeño comercial de RappiPlus mediante la integración de datos de pedidos, del catálogo de productos y de gastos de marketing. El objetivo es identificar oportunidades de crecimiento, evaluar la rentabilidad del negocio y generar recomendaciones basadas en datos.

## Objetivos

- Evaluar la rentabilidad del negocio.
- Analizar el comportamiento de ventas.
- Identificar pérdidas dentro del funnel de conversión.
- Medir la retención de usuarios mediante cohortes.
- Validar cambios mediante pruebas estadísticas.
- Diseñar un dashboard ejecutivo para la toma de decisiones.
